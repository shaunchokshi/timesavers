# timesavers
projects to make other self-imposed projects easier, which will likely spawn new problems requiring projects to make solutions.

### there are submodules which pull in zsh custom plugins to include in oh-my-zsh
### to clone with all the submodules:
```git clone --recursive -j8 https://github.com/shaunchokshi/timesavers/```

##### or if you already have the main repo cloned, update to download the submodules:
```git submodule init && git submodule update```
